# Firebase cloud messaing
> Firebase Admin SDK를 이용한 푸쉬 전송 예제입니다.
> 
> resources/fcm 디렉토리에 admin sdk를 추가하여 사용하시면 됩니다.

✔️ 테스트 환경
* Spring Boot 2.7.5 
* JDK 1.8 
* Gradle

✔️ 프로젝트 구조
- - -
```
.
.
└── src/
    └── main/
        ├── java/
        │   └── com/
        │       └── example/
        │           └── firebasepush/
        │               ├── api/
        │               ├── exception/
        └── resources/
            ├── fcm            
```
